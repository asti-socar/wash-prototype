# 파트너 관리

## 1. 개요
- **화면명**: 파트너 관리
- **목적**: 계약된 파트너사의 기본 정보와 단가 정책을 관리
- **주요 기능**: 파트너 목록 조회, 신규 등록, 기본 정보 수정, 단가 정책 관리

## 2. 화면 구성 및 기능

### 2.1 리스트
- **[신규 파트너 등록]** 버튼으로 등록 Drawer 호출
- **페이지네이션**: 40건
- **정렬**: 헤더 클릭 시 오름차순/내림차순

#### 컬럼
| 헤더명 | 정렬 | 비고 |
| :--- | :---: | :--- |
| 파트너 이름 | O | |
| 대표자 | O | |
| 계약 상태 | O | Badge: 운영중(파란색), 종료(빨간색) |
| 휴대전화 | O | |
| 배정 존 개수 | O | `{N}개` 형식 |

### 2.2 상세/등록 (Drawer)
행 클릭 시 상세 조회. 2개 탭으로 구성.

#### ① 기본 정보 탭
| 필드 | 타입 | 비고 |
| :--- | :--- | :--- |
| 파트너 이름 | Input | 필수 |
| 대표자 | Input | 필수 |
| 사업자번호 | Input | |
| 주소 | Input | |
| 휴대전화 | Input | |
| 이메일 | Input | |
| 계약 시작일 | DatePicker | |
| 계약 종료일 | DatePicker | |
| 계약 상태 | Select | 운영중 / 종료 |

- Footer: [수정하기] (기존 파트너) 또는 [등록하기] (신규) + [닫기]

#### ② 단가 정책 관리 탭
파트너별 오더 구분/세차 유형 조합에 따른 단가를 관리합니다.

**적용 중 단가 정책**: 적용 시작일이 오늘 이전인 정책 (최신순 정렬)

**예정된 단가 정책**: 적용 시작일이 오늘 이후인 정책 (날짜순 정렬)

**정책 추가 폼**:
- 오더 구분 (긴급/정규/변경/수시/특별)
- 세차 유형 (내외부/내부/외부/특수/협의/라이트/기계세차)
- 단가 (숫자)
- 적용 시작일 (DatePicker)
- confirm 후 추가

**정책 삭제**: 각 행의 삭제 아이콘 클릭 → confirm 후 삭제

- Footer: [닫기]만 표시 (단가 탭에서는 수정 버튼 없음)
