# AI 모델 정책 관리

## 1. 개요
- **화면명**: AI 모델 정책 관리
- **목적**: AI 모델의 오더 생성 기준과 고객 피드백 정책을 관리
- **주요 기능**: 오염도 레벨 설정, 긴급 오더 피드백 단어 사전 관리

## 2. 화면 구성 및 기능

### 2.1 공통 UI
- 상단 우측에 [정책 저장] / [취소] 버튼
- 두 설정 중 하나라도 변경되면 버튼 활성화, 변경 없으면 비활성화
- 각 카드 우측에 "변경됨" 표시 (해당 설정이 변경된 경우)

### 2.2 오염도 기반 오더 생성 레벨
CV 모델이 판별한 오염도 레벨에 따라 세차 오더를 생성하는 기준을 설정합니다.

- **입력 방식**: PillTabs (Level 0 ~ Level 3)
- **기본값**: Level 2
- **동작**: 설정된 레벨 이상의 오염도를 가진 차량에 대해 오더 생성
- **안내 문구**: 현재 설정 레벨 및 변경 전 레벨 표시

### 2.3 긴급 오더 피드백 단어 사전
고객 피드백에 포함된 경우 긴급 세차 오더를 발행할 단어를 설정합니다.

- **단어 추가**: Input + [추가] 버튼. Enter 키로도 추가 가능. 자동으로 `#` 접두사 부여.
- **단어 삭제**: 각 Chip의 X 버튼 클릭 시 삭제
- **중복 방지**: 동일 단어 중복 등록 불가
- **변경 표시**: 새로 추가된 단어는 파란색 Chip으로 구분
- **빈 상태**: 등록된 단어가 없으면 "등록된 단어가 없습니다." 표시
