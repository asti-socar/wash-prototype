# 대시보드

## 1. 개요
- **화면명**: 세차 운영 대시보드
- **목적**: 전체 세차 오더 현황 실시간 모니터링, 리스크 오더 사전 감지/개입, 파트너 성과 추적
- **데이터 갱신**: 수동 (새로고침 버튼), 조회 기준일 오늘 고정

## 2. 화면 구성 및 기능

### 2.1 헤더
| 요소 | 설명 |
| :--- | :--- |
| 타이틀 | "세차 운영 대시보드" |
| 서브타이틀 | "실시간 현황 모니터링 및 리스크 관리" |
| 갱신 시간 | "마지막 갱신: N초 전" 또는 "N분 전" |
| 새로고침 버튼 | RefreshCw 아이콘 + "새로고침", 클릭 시 전체 데이터 갱신 |

### 2.2 [영역 1] 오더 현황

전체 오더의 상태별 분포를 **도넛 차트 + 4개 상태 카드**로 시각화합니다.

#### 도넛 차트
- 4개 상태: 수행 대기(발행+예약), 수행 중, 완료, 취소
- 차트 중앙: 전체 오더 수 + "전체" 라벨
- 범례: 차트 우측 세로 배치 (`● 상태명: 건수`), 클릭 시 오더 관리로 이동

#### 상태 카드 (4개)
| 카드 | 메인 수치 | 하위 항목 | 색상 |
| :--- | :--- | :--- | :--- |
| 수행 대기 | 발행 + 예약 합계 | 발행: N건, 예약: N건 | #7BA3C9 |
| 수행 중 | 수행 중 건수 | - | #E8C47C |
| 완료 | 완료 건수 | 적시수행: N건, 지연수행: N건 | #7BC9A8 |
| 취소 | 취소 건수 | 변경, 미예약, 노쇼, 수행원, 우천 (유형별 건수) | #D98E8E |

- 각 카드/하위 항목 클릭 시 해당 필터로 오더 관리 페이지 이동

### 2.3 [영역 2] 리스크 관리

리스크 유형별 오더 현황을 **탭 + 도넛 차트 + 4개 상태 카드**로 모니터링합니다.

#### 탭 구성
| 탭명 | orderType 필터값 | 배지 |
| :--- | :--- | :--- |
| 위생장애 | "위생장애" | 건수 |
| 고객피드백(ML)_긴급 | "고객 피드백(ML)_긴급" | 건수 |
| 초장기미세차 | "초장기 미세차" | 건수 |

- 선택 탭: 파란 텍스트(#0052CC) + 언더라인 + 파란 배지
- 미선택 탭: 회색 텍스트(#6B778C) + 회색 배지

#### 리스크 도넛 차트
- 리스크 유형별 오더만 표시 (일반 오더 제외)
- 차트 중앙: 리스크 총합 + "리스크" 라벨

#### 탭별 상태 카드
- 오더 현황과 동일한 4개 카드 구조 (수행 대기, 수행 중, 완료, 취소)
- 취소 카드: 0건인 유형은 숨김
- 각 항목 클릭 시 `{ status, orderType }` 조합으로 오더 관리 이동

### 2.4 [영역 3] 파트너 수행 관리

파트너별 오더 현황 테이블입니다.

#### 컬럼 정의
| 컬럼 | 집계 방식 | 클릭 시 필터 |
| :--- | :--- | :--- |
| 파트너 | - | `{ partner }` |
| 담당 | 전체 오더 수 | `{ partner }` |
| 대기 | 발행+예약 오더 수 | `{ partner, status: "발행" }` |
| 수행중 | 수행 중 오더 수 | `{ partner, status: "수행 중" }` |
| 리스크 | 리스크 오더 수 (위생장애+ML긴급+초장기미세차) | `{ partner, orderType: "위생장애" }` |
| 지연 | 수행 중 + elapsedDays > 7 | `{ partner, status: "수행 중" }` |

- **기본 정렬**: 담당 오더 수 내림차순

#### 경고 표시
| 지표 | 주의 (노랑 #E8C47C) | 위험 (빨강 #D98E8E) |
| :--- | :--- | :--- |
| 대기 | ≥ 7 | ≥ 10 |
| 리스크 | ≥ 3 | ≥ 5 |
| 지연 | ≥ 1 | ≥ 3 |

- 경고 시 수치 앞에 색상 원형 인디케이터 표시

#### 리드타임 및 지연 정책 안내

> **리드타임 (Lead Time)**
> 오더 발행부터 완료까지의 소요 시간입니다. 참조 오더 관계(취소→재발행 체인)가 있는 경우, 연결된 모든 구간을 합산합니다.
> 단, 파트너 무과실 취소(우천, 노쇼 등) 구간과 긴급/장애로 별도 발행된 오더는 리드타임 산출에서 제외됩니다.
> 파트너 단위 평균 리드타임으로 집계하며, 경고 기준은 7일(주의) / 10일(위험)입니다.
>
> **오더 지연**
> 발행 후 72시간이 경과하도록 완료되지 않은 오더입니다. 대시보드에서는 수행 중 상태에서 경과일 7일 초과 시 지연으로 표시하며, 경고 기준은 1건(주의) / 3건(위험)입니다.
>
> 상세 정의는 오더 관리 스펙(orders.md) 섹션 2.4~2.5를 참조하세요.

### 2.5 [영역 4] 시계열 트렌드 그래프

2열 레이아웃으로 최근 7일 추이를 시각화합니다.

#### A. 일자별 정규 오더 생성량 (막대 그래프)
- X축: 날짜 (MM/DD), Y축: 건수
- 막대 색상: rgba(123, 163, 201, 0.8)
- 호버 시 툴팁: 날짜, 정규 오더 N건

#### B. 일별 리스크 유형별 추이 (라인 차트)
| 라인 | 색상 |
| :--- | :--- |
| 위생장애 | rgba(217, 142, 142, 0.8) |
| ML긴급 | rgba(232, 196, 124, 0.8) |
| 초장기미세차 | rgba(123, 163, 201, 0.8) |

- 범례: 차트 상단 우측, 원형 아이콘
- 호버 시 툴팁: 날짜, 유형별 건수

## 3. 색상 팔레트

차트 색상은 80% 투명도 적용, 카드 숫자 및 범례는 100% 색상 유지.

| 색상명 | HEX | RGBA (80%) | 용도 |
| :--- | :--- | :--- | :--- |
| Steel Blue | #7BA3C9 | rgba(123, 163, 201, 0.8) | 수행 대기, 초장기미세차 |
| Warm Beige | #E8C47C | rgba(232, 196, 124, 0.8) | 수행 중, ML긴급 |
| Soft Teal | #7BC9A8 | rgba(123, 201, 168, 0.8) | 완료 |
| Dusty Rose | #D98E8E | rgba(217, 142, 142, 0.8) | 취소, 위생장애 |
