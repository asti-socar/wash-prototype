# 존 배정 관리

## 1. 개요
- **화면명**: 존 배정 관리
- **목적**: 전체 존 기준으로 파트너 배정 현황을 관리. 미배정 존을 식별하고 파트너를 배정
- **주요 기능**: 존 목록 조회, 배정 상태 필터링, 파트너 배정/변경/해제, 대량 존 배정(CSV/Excel)

## 2. 데이터 정책

### 2.1 데이터 관계
- **존 → 파트너**: 1:2 (하나의 존에 현장 파트너 1개 + 입고 파트너 1개 배정)
- **파트너 → 존**: 1:N (하나의 파트너가 여러 존 담당 가능)

### 2.2 파트너 유형
| 유형 | 설명 | 현재 파트너 |
| :--- | :--- | :--- |
| 현장 파트너 | 오더의 파트너 유형이 "현장"인 경우 배정 | 강남모빌리티(P-001), 미션핸들코리아(P-003) |
| 입고 파트너 | 오더의 파트너 유형이 "입고"인 경우 배정 | 수원카케어(P-002) |

### 2.3 배정 상태 변경
- **배정**: 미배정 존에 파트너를 선택 (현장/입고 각각)
- **변경**: 이미 배정된 존의 파트너를 변경 (현장/입고 각각)
- **해제**: 배정된 파트너를 "미배정"으로 변경 (전체/현장만/입고만 선택 가능)

## 3. 화면 구성 및 기능

### 3.1 리스트
- **페이지네이션**: 40건
- **기본 정렬**: 존 ID 내림차순. 정렬 가능 컬럼에 한해 헤더 클릭 시 오름차순/내림차순 토글. 정렬 가능 컬럼은 헤더에 화살표 아이콘(↕)으로 표시.
- **검색**: 존 ID 또는 존 이름
- **필터**: 지역1, 지역2(지역1 선택 전 비활성화), 현장 파트너, 입고 파트너

#### 컬럼
| 헤더명 | 정렬 | 비고 |
| :--- | :---: | :--- |
| 존 ID | | |
| 지역1 | | |
| 지역2 | | |
| 존 이름 | | |
| 존 유형 | | 현장세차존, 주기세차존, 현장세차 불가존 |
| 차량 대수 | O | |
| 현장 파트너 | | 파트너명 또는 Badge(미배정) |
| 입고 파트너 | | 파트너명 또는 Badge(미배정) |
| (액션) | | Pencil(수정) + Trash2(배정 해제) 아이콘 |

### 3.2 존 배정 상세 (Drawer)
- **타이틀**: 상세 시 `존 배정 상세 - {존ID}`, 수정 시 `존 배정 수정 - {존ID}`
- 행 클릭 시 상세 조회(읽기 모드). Pencil 아이콘 클릭 시 바로 수정 모드로 열림.
- **Footer**:
  - 읽기 모드: [닫기] ... [수정] [삭제]
  - 편집 모드: [취소] [저장]

#### A. 존 정보
존 ID, 존 이름, 지역(지역1/지역2), 전체 주소, 존 유형 (모두 읽기 전용)

#### B. 차량 정보
차량 대수 (읽기 전용)

#### C. 배정된 파트너
- **현장 파트너**
  - 읽기 모드: 파트너명 또는 "미배정" Badge 표시
  - 수정 모드: Select 드롭다운으로 현장 파트너 선택/변경. 옵션: "미배정" + 현장 파트너 목록.
- **입고 파트너**
  - 읽기 모드: 파트너명 또는 "미배정" Badge 표시
  - 수정 모드: Select 드롭다운으로 입고 파트너 선택/변경. 옵션: "미배정" + 입고 파트너 목록.

### 3.2.1 수정 확인 팝업 (SaveConfirmModal)
- 저장 클릭 시 변경된 파트너 정보를 비교하여 변경 내역(From → To)을 팝업으로 표시
- 현장 파트너, 입고 파트너 각각의 변경 사항 표시
- 변경 사항이 없으면 팝업 없이 편집 모드 종료
- [확인] 클릭 시 저장 반영 후 Drawer 닫힘, [취소] 클릭 시 팝업만 닫힘
- z-index: 1100 (Drawer 위에 표시)

### 3.2.2 배정 해제 (DeleteConfirmModal)
- 리스트 Trash2 아이콘 또는 상세 Drawer [삭제] 버튼 클릭 시 배정 해제 확인 팝업 표시
- 두 파트너 모두 배정된 경우 해제 범위 선택:
  - 전체 해제: 현장 + 입고 파트너 모두 해제
  - 현장 파트너만 해제
  - 입고 파트너만 해제
- 하나만 배정된 경우: 해당 파트너만 해제 (범위 선택 없음)
- [해제] 클릭 시 선택된 범위의 파트너를 null로 변경 (미배정 상태)

#### D. 현장세차 수행원 / 입고세차 수행원
- 각 파트너 유형별로 별도 카드에 수행원 표시
- 해당 파트너가 배정된 존에서만 표시 (소속 파트너 + 배정 존 일치 수행원)
- **컬럼**: 수행원 ID, 이름, 벌점 (0이면 회색, 1 이상이면 빨간 배지)
- 파트너 미배정: "현장/입고 파트너가 배정되지 않은 존입니다." 안내
- 수행원 없음: "배정된 수행원이 없습니다." 안내

### 3.3 대량 존 배정 (Drawer)
[대량 존 배정] 버튼 클릭 시 드로어 열림.

- **양식 다운로드**: CSV 템플릿 (`zoneId,onsitePartnerId,inshopPartnerId`)
- **파일 업로드**: 클릭 또는 드래그앤드롭. CSV(.csv), Excel(.xlsx, .xls) 지원.
- **검증 결과**: 배정 가능 건수 + 실패 건(사유 포함) 표시
  - 미리보기 테이블: 존 ID, 현장 파트너, 입고 파트너 3컬럼
- **배정 적용**: [N건 배정 적용] 버튼 → confirm 후 일괄 반영

#### 검증 로직
| 검증 항목 | 실패 메시지 |
| :--- | :--- |
| 존 ID 누락 | 존 ID가 비어있습니다. |
| 존재하지 않는 존 ID | 존재하지 않는 존 ID입니다. |
| 유효하지 않은 현장 파트너 ID | 현장 파트너 ID({ID})가 유효하지 않습니다. |
| 유효하지 않은 입고 파트너 ID | 입고 파트너 ID({ID})가 유효하지 않습니다. |
