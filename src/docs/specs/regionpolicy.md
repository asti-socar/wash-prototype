# 지역 정책 관리

## 1. 개요
- **화면명**: 지역 정책 관리
- **목적**: 시/도(Region1) 및 시군구(Region2) 단위의 공통 세차 정책을 설정. 존에 개별 정책이 없으면 해당 지역 정책이 기본값으로 적용.
- **주요 기능**: 지역1/지역2 정책 탭 분리 조회, 인라인 수정

## 2. 화면 구성 및 기능

### 2.1 탭 구성
- **지역1 정책 (시도)**: 시/도 단위 정책 목록
- **지역2 정책 (시군구)**: 시군구 단위 정책 목록

### 2.2 관리 필드
| 필드명 | 설명 |
| :--- | :--- |
| 지역 ID | 지역1 ID 또는 지역2 ID |
| 시도 / 시군구 | 지역 명칭 |
| 주기세차(일) | 주기세차 오더 발행 기준 일수 |
| 라이트세차 | Y/N (Badge: Y=녹색, N=회색) |

### 2.3 지역1 정책 (시도) 탭
- **목록 조회**: 시도별 정책 리스트. 페이지네이션 없이 전체 노출.
- **수정 모드**: [수정] 버튼 → 주기세차(일): Input, 라이트세차: Select로 전환
- **저장/취소**: [저장] 클릭 시 반영, [취소] 클릭 시 변경 파기

### 2.4 지역2 정책 (시군구) 탭
- **필터**: 시도 Select로 특정 시도의 시군구만 필터링
- **검색**: 시군구 이름 검색. 콤마(,) 또는 공백으로 여러 시군구 동시 검색 가능.
- **수정 모드**: 지역1과 동일한 인라인 수정 기능
- **페이지네이션**: 100건
- **필터 연동**: 필터/검색 상태에서 수정 시 화면에 보이지 않는 데이터 유실 방지
