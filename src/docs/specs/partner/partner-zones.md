# 수행원 배정

## 1. 개요
- **화면명**: 수행원 배정
- **목적**: 자사에 배정된 존별로 담당 수행원을 조회하고 배정/변경
- **주요 기능**: 존 목록 조회/필터링, 수행원 이름 검색, 존별 수행원 배정 수정
- **권한**: 수행원 배정/해제만 가능 — 존 정보, 파트너 배정, 존 삭제 불가

### 인터널 존 배정 관리 대비 차이점
| 항목 | 인터널 (존 배정 관리) | 파트너 (수행원 배정) |
| :--- | :--- | :--- |
| 화면명 | "존 배정 관리" | "수행원 배정" |
| 서브타이틀 | "존별 파트너 배정 현황을 관리합니다." | "{partnerName}가 담당하는 존별 수행원 배정을 관리합니다." |
| 데이터 범위 | 전체 존 (50건) | 자사가 현장 파트너로 배정된 존만 표시 |
| 대량 존 배정 | CSV/Excel 업로드 Drawer | **제거** |
| 현장/입고 파트너 필터 | 필터 2개 (현장, 입고) | **제거** |
| 수행원 이름 검색 | 없음 | **추가** (필터 영역에 검색 필드) |
| 목록 컬럼 — 존 유형 | 존 유형 컬럼 표시 | **제거** |
| 목록 컬럼 — 파트너 | 현장 파트너, 입고 파트너 2개 컬럼 | **제거** (배정 수행원 컬럼으로 대체) |
| 목록 컬럼 — 배정 수행원 | 없음 | **추가** (수행원 이름 콤마 구분 표시) |
| 목록 액션 — 배정 해제 | Trash2 아이콘으로 배정 해제 | **제거** |
| Drawer 존 정보 | 존 유형 포함 | 존 유형 **제거**, 운영 시간·주차 유형 추가 |
| Drawer 파트너 변경 | 현장/입고 파트너 Select 드롭다운 | **제거** |
| Drawer 수행원 수정 | 없음 (조회만) | **추가** (검색+선택 방식으로 수행원 배정) |
| Drawer Footer 삭제 버튼 | [삭제] 버튼 표시 | **제거** |
| 최소 수행원 제약 | 없음 | 존당 최소 1명 이상 배정 필수 |

---

## 2. 데이터 정책

### 2.1 데이터 범위
- 자사(`currentPartner.partnerId`)가 **현장 파트너(`assignedOnsitePartnerId`)**로 배정된 존만 조회 대상
- 수행원은 자사 소속(`partnerId === currentPartner.partnerId`) 수행원만 표시

### 2.2 수행원-존 관계
- **수행원 → 존**: 1:N (한 수행원이 여러 존 담당 가능)
- **존 → 수행원**: 1:N (한 존에 여러 수행원 배정 가능)
- **제약**: 모든 존에는 최소 1명 이상의 수행원이 배정되어야 함

## 3. 화면 구성 및 기능

### 3.1 헤더
| 요소 | 설명 |
| :--- | :--- |
| 타이틀 | "수행원 배정" |
| 서브타이틀 | "{partnerName}가 담당하는 존별 수행원 배정을 관리합니다." |

- 인터널의 **[대량 존 배정]** 버튼은 제거

### 3.2 검색 및 필터

상단 카드 영역에서 존 리스트를 필터링하며, 선택된 조건은 하단에 **칩(Chip)** 형태로 노출됩니다.

#### 검색 조건
| 구분 | 항목 | 타입 | 동작 정의 |
| :--- | :--- | :--- | :--- |
| **검색** | 검색항목 | 선택(Select) | 존 ID / 존 이름 |
| **검색** | 검색어 | 텍스트 검색 | 검색 아이콘 포함. 부분 일치 검색 |
| **필터** | 지역1 | 선택(Select) | 전체 + 자사 배정 존의 고유 지역1 |
| **필터** | 지역2 | 선택(Select) | 지역1 선택 시 활성화. 전체 + 해당 지역1 하위 지역2 |
| **검색** | 수행원 이름 | 텍스트 검색 | 검색 아이콘 포함. 해당 수행원이 배정된 존만 필터링 |

#### UI 동작
- **필터 칩**: 적용된 조건이 칩으로 나열, `X` 버튼으로 개별 해제
- **설정 초기화**: 모든 입력값 및 칩 제거
- **지역2 비활성화**: 지역1 미선택 시 지역2는 disabled + 회색 배경

### 3.3 존 목록

#### 컬럼 (7개 + 액션)
| 헤더명 | 데이터 정의 |
| :--- | :--- |
| 존 ID | 존 고유 식별자 |
| 지역1 | 시/도 단위 |
| 지역2 | 구/군/시 단위 |
| 존 이름 | 존 이름 |
| 차량 대수 | `{N}대` 포맷 |
| 배정 수행원 | 배정된 수행원 이름 콤마 구분 표시. 미배정 시 Badge(warn) "미배정" |
| (액션) | Pencil 아이콘 — 클릭 시 수정 모드 Drawer 열림 |

- **기본 정렬**: 존 ID 내림차순
- **페이지네이션**: 40건
- **결과 카운트**: "필터된 결과 N건 / 전체 N건" 텍스트 표시

### 3.4 수행원 배정 상세/수정 (Drawer)

목록의 행 클릭 시 읽기 모드로 Drawer 오픈. Pencil 아이콘 클릭 시 수정 모드로 직접 오픈.

- **타이틀**: 읽기 시 `수행원 배정 상세 - {존ID}`, 수정 시 `수행원 배정 수정 - {존ID}`
- **서브타이틀**: 존 이름

#### Footer
| 모드 | 버튼 |
| :--- | :--- |
| 읽기 | [닫기] ... [수정] |
| 수정 | [취소] [저장] |

#### A. 존 정보 카드
| 필드 | 설명 |
| :--- | :--- |
| 존 ID | 읽기 전용 |
| 존 이름 | 읽기 전용 |
| 지역 | `{지역1} / {지역2}` 읽기 전용 |
| 전체 주소 | 읽기 전용 |
| 운영 시간 | 읽기 전용 |
| 주차 유형 | 읽기 전용 |

#### B. 차량 정보 카드
| 필드 | 설명 |
| :--- | :--- |
| 차량 대수 | `{N}대` 읽기 전용 |

#### C. 배정 수행원 카드

**읽기 모드**
- 카드 헤더에 배정 인원 Badge 표시 (`{N}명`, 1명 이상이면 Green, 0명이면 Warn)
- 수행원 테이블: 수행원 ID, 이름, 담당 존(N개), 벌점 (0이면 회색, 1 이상이면 위험 Badge)
- 수행원 없음: "배정된 수행원이 없습니다." 안내

**수정 모드 — 검색+선택 방식**
- **배정된 수행원 태그**: 현재 배정된 수행원을 파란 태그(chip)로 표시. 각 태그에 `X` 버튼으로 개별 해제 가능.
  - 마지막 1명은 해제 불가 (최소 1명 제약). 시도 시 에러 메시지 표시.
- **수행원 검색**: 텍스트 입력 시 이름 또는 ID 부분 일치로 미배정 수행원을 드롭다운에 표시.
  - 드롭다운 항목: 수행원 이름, ID, 담당 존 수, 벌점 정보.
  - 항목 클릭 시 즉시 배정 태그에 추가되고 드롭다운 닫힘.
  - 검색 입력창 포커스 시 드롭다운 자동 열림 (전체 미배정 수행원 목록).
  - 드롭다운 외부 클릭 시 자동 닫힘.
  - 모든 수행원 배정 완료 시: "추가할 수행원이 없습니다." 안내.
  - 검색 결과 없음 시: "검색 결과가 없습니다." 안내.

### 3.4.1 수정 확인 팝업 (SaveConfirmModal)
- 저장 클릭 시 변경 내역(추가된 수행원 / 제거된 수행원)을 팝업으로 표시
  - 추가된 수행원: 초록 배경 카드에 이름 목록
  - 제거된 수행원: 분홍 배경 카드에 이름 목록
- 변경 사항이 없으면 팝업 없이 편집 모드 종료
- [확인] 클릭 시 저장 반영 후 Drawer 닫힘, [취소] 클릭 시 팝업만 닫힘

## 4. 운영 시나리오

**존별 수행원 현황 조회**: 수행원 배정 진입 → 지역 필터 또는 수행원 이름 검색으로 존 필터링 → 목록에서 배정 수행원 확인

**특정 존 수행원 변경**: 존 행 클릭 또는 Pencil 아이콘 → Drawer 수정 모드 → 불필요한 수행원 태그 `X` 클릭으로 제거 → 검색창에서 새 수행원 검색 후 선택 → 저장 → 변경 내역 확인 후 확인

**참고**: 존 정보 수정, 파트너 배정 변경, 존 배정 해제는 인터널 어드민에서만 수행 가능합니다. 파트너 어드민에서는 수행원 배정만 관리 가능합니다.
