# 파트너 담당자 관리

## 1. 개요
- **화면명**: 파트너 담당자 관리
- **목적**: 파트너사 소속 담당자의 정보를 등록, 수정, 삭제
- **주요 기능**: 담당자 목록 조회/필터링, 담당자 등록/수정/삭제(Drawer)

## 2. 화면 구성 및 기능

### 2.1 리스트
- **[담당자 등록]** 버튼으로 등록 Drawer 호출
- **필터**: 파트너 이름 Select
- **페이지네이션**: 40건
- **기본 정렬**: 담당자 ID 내림차순. 헤더 클릭 시 오름차순/내림차순 토글.

#### 컬럼
담당자 ID, 파트너 이름, 직무, 이름, 휴대전화, 이메일

#### 리스트 마스킹
목록 조회 시 개인정보 마스킹 적용. 상세(Drawer)에서는 원본 표시.

| 필드 | 규칙 | 예시 |
| :--- | :--- | :--- |
| 이름 | 2글자: 마지막 마스킹, 3글자: 가운데 마스킹, 4글자+: 처음·마지막 제외 마스킹 | 김* / 김*당 / 김**저 |
| 휴대전화 | 가운데 자릿수 마스킹 | 010-\*\*\*\*-5678 |
| 이메일 | 로컬 첫 3자 + 마스킹, 도메인 첫 1자 + 마스킹 | kim\*\*@g\*\*\*\*\*\*\*\*\* |

### 2.2 등록/수정 (Drawer)
행 클릭 시 수정 모드, [담당자 등록] 클릭 시 신규 등록 모드.

모든 입력 필드는 필수입니다.

#### 카드 1: 파트너 정보
| 필드 | 타입 | 비고 |
| :--- | :--- | :--- |
| 파트너 | Select | 전체 파트너 목록에서 선택 |
| 직무 | Select | 세차 담당 / 행정 담당 / 기타 |

#### 카드 2: 계정 정보
| 필드 | 타입 | 비고 |
| :--- | :--- | :--- |
| 이름 | Input | |
| 이메일 | Input | |
| 휴대전화 | Input | |

- **유효성 검사**: 미입력 항목 존재 시 경고 표시
- **Footer**: [등록하기]/[수정하기] + [취소]. 수정 모드에서는 좌측에 [삭제] 버튼 (confirm 후 삭제).
