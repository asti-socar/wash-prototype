# 미션 관리

## 1. 개요
- **화면명**: 미션 관리
- **목적**: 차량에 사전 할당된 특수 과업을 관리하고, 세차 오더 발행 시 해당 정보를 자동 연동하여 수행 여부를 추적합니다.

## 2. 기능 명세

### 2.1 미션 리스트
세차 오더가 발행되기 전후의 미션 현황을 통합 조회합니다.

- **검색 및 필터**
  - **통합 검색**: 차량번호, 미션 ID, 존이름, 존 ID 검색.
  - **상태 필터**: 대기(수행 전), 수행완료 중 선택.
  - **기간 필터**: 미션이 등록된 날짜 범위를 지정하여 조회.

- **목록 컬럼 구성**
  - **미션 ID**: 고유 번호.
  - **차량번호**: 대상 차량.
  - **존이름**: 차량이 소속된 구역.
  - **미션 내용**: 수행해야 할 과업 (예: 스티커 제거, 내부 집중 케어 등).
  - **상태**: 현재 진행 단계 (대기 또는 수행완료).
  - **연결된 오더**: 해당 미션이 포함된 세차 오더 (클릭 시 오더 상세 화면으로 이동).
  - **등록일**: 미션이 처음 등록된 일시.

### 2.2 미션 상세
리스트의 항목을 클릭하면 우측 창(Drawer)을 통해 상세 내역을 확인합니다.

- **구성 섹션**
  1. **기본 정보**: 차량의 세부 정보(차종, 연료 등) 및 등록된 미션의 상세 설명.
  2. **오더 연동 정보**: 현재 미션을 수행하고 있는 세차 오더의 진행 상태와 담당 수행원 정보.

- **상태 자동 업데이트**
  - 연결된 세차 오더가 최종 '수행 완료' 처리되면, 해당 미션의 상태도 별도 조작 없이 자동으로 '수행완료'로 변경됩니다.

### 2.3 미션 관리 동작
- **미션 등록**: 화면 상단의 버튼을 통해 신규 미션을 생성합니다. 등록된 미션은 차기 세차 오더 발행 시 자동으로 포함됩니다.
- **미션 삭제**: 더 이상 유효하지 않은 미션은 상세 화면에서 삭제할 수 있습니다. 삭제 시에는 반드시 **삭제 사유**를 입력해야 하며, 처리 내역은 시스템에 기록됩니다.
